# AMANO Interiors Portfolio

An ultra-premium, interactive single-page portfolio application for **AMANO Interiors**, a bespoke luxury modular kitchen and home design studio based in Bengaluru.

## 🚀 Live Demo
Deploy your own version instantly on Vercel or Netlify.
https://amanointeriors.vercel.app

---

## ✨ Key Features

- **Luxury Design & Animations**: Smooth momentum scrolling powered by **Lenis** and physics-based cursor coordinates 3D tilt translations powered by **Framer Motion**.
- **Interactive Header Navigation**: Transparent navigation bar overlay at the top of the viewport that transitions into a fixed glassmorphic blurred header upon scroll. Features a hover-based services dropdown and layout-linked active dot indicators.
- **Bento Services Grid**: A responsive 3-column bento grid detailing Modular Kitchens, Wardrobes, and Complete Interiors with mouse hover perspective tilt effects.
- **Consultation Form**: Frosted glassmorphic contact form featuring floating labels, timezone (IST) clock tracker, and bespoke callback booking validation.
- **Symmetric Footer**: Comprehensive multi-column luxury footer mirroring the top navbar's clean styling, equipped with interactive Google Maps redirects, social icons, and a stylized serving areas map.
- **Action Sidebar**: Fixed right sidebar housing direct Call, WhatsApp, and scroll progress tracker widgets.

---

## 🛠️ Tech Stack

- **Framework**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Scrolling**: [Lenis Smooth Scroll](https://lenis.darkroom.engineering/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Vanilla CSS Variables (Tactile Paper Noise & Architectural Blueprint Overlays)

## 💻 Local Development

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/amano-interiors.git
   cd amano-interiors
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the local development server:**
   ```bash
   npm run dev
   ```
   Open your browser and navigate to `http://localhost:3000/`.

4. **Compile for production build:**
   ```bash
   npm run build
   ```
   The compiled static files will be placed inside the `dist/` directory, ready to be served.
